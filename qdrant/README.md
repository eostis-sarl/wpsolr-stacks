# WPSOLR Qdrant Stack

![Qdrant](https://img.shields.io/badge/Qdrant-0093DD?logo=qdrant&logoColor=white)

This directory contains a **Docker Compose stack** for running a Qdrant based backend with WPSOLR.  
It’s intended for **local development and testing** with WordPress + WPSOLR.
